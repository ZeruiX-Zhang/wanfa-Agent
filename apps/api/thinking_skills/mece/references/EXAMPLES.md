# MECE 分解示例

## 示例 1：SaaS 产品用户流失原因分解

**问题**：为什么付费用户在试用期结束后不续费？

**MECE 分解**：

```
用户不续费原因
├── 产品价值不足
│   ├── 核心功能未满足需求
│   ├── 竞品功能更强
│   └── 使用频率太低感知不到价值
├── 价格因素
│   ├── 超出预算
│   ├── 性价比不如竞品
│   └── 付费方案不灵活
├── 体验障碍
│   ├── 上手成本高
│   ├── 关键流程卡顿/报错
│   └── 缺少培训和支持
└── 外部因素
    ├── 业务方向调整不再需要
    ├── 决策人变更
    └── 公司预算冻结
```

**ME 检查**：✅ 四个分支（产品/价格/体验/外部）互不重叠
**CE 检查**：✅ 覆盖了用户决策的所有维度

## 示例 2：电商大促 GMV 增长策略

**问题**：如何在双 11 实现 GMV 同比增长 30%？

**MECE 分解**（按 GMV = 流量 × 转化率 × 客单价）：

```
GMV 增长路径
├── 提升流量
│   ├── 站外投放（信息流/搜索/KOL）
│   ├── 站内流量（搜索权重/推荐位）
│   └── 私域召回（短信/Push/社群）
├── 提升转化率
│   ├── 商品详情页优化
│   ├── 促销机制简化
│   └── 支付流程缩短
└── 提升客单价
    ├── 满减/凑单引导
    ├── 套装组合推荐
    └── 会员专属加价购
```

**ME 检查**：✅ 流量/转化/客单价三者数学上互斥
**CE 检查**：✅ GMV = 三者乘积，穷尽所有增长杠杆
