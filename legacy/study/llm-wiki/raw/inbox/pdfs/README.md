# raw/inbox/pdfs/

放在这里的内容:任意 PDF——论文、书籍、白皮书、行业报告、说明书。

## 怎么放

直接把 PDF 拖到这个文件夹。**不用改名,不用提前分类。**

## Claude 处理时会怎么做

1. **保留原 PDF**(永远不删、不改、不替换)。
2. 尝试提取文字:
   - 文字版 PDF → 提取成功,生成 Markdown 副本到 `raw/papers/` 或 `raw/books/`
   - 扫描版 PDF → 文字提取失败,标记为 `needs-ocr`,写进状态页
3. 文字够用时,执行 Ingest Workflow,生成 `wiki/sources/` 摘要页
4. 文字不够时,**绝不假装看懂**,只创建一个占位 source 页,等你后续 OCR 或自己抄关键段落

## 大文件提醒

单个 PDF > 50MB 时,处理会慢且可能失败。建议你自己先用阅读器导出关键章节为 PDF 再丢进来——不强制,但更顺。
