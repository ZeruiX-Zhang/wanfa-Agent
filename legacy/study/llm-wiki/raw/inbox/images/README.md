# raw/inbox/images/

放在这里的内容:截图、照片、图表、白板照片、手稿照片。

## 怎么放

直接拖图片进来。命名随意,Claude 处理时会:
1. 用视觉识别看图,判断主题
2. 重命名为 `YYYY-MM-DD-描述.png`
3. 移到 `raw/assets/`
4. 如果图里有重要文字(架构图、公式、代码截图),会做 OCR/识别 并写一份说明文件

## 搭配使用建议

如果这张图是某段笔记的配图,建议同时:
1. 把图丢进 inbox/images/
2. 把对应的文字笔记粘到 `../notes.md`
3. 在文字里写一句 "见图: filename.png"

Claude 处理时会把它们关联起来。
