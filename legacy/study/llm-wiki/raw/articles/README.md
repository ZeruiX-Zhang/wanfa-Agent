# raw/articles/

放在这里的内容:网页文章、博客、新闻、长帖等。

规则:
- 文件名建议格式: `YYYY-MM-DD-标题简写.md` 或 `YYYY-MM-DD-标题简写.html`
- 一旦放进来就不再修改/重命名/删除(由 wiki 层处理脏数据)
- 可以是纯文本、Markdown、HTML、PDF 等任何格式

如何放入:
1. 直接把文件复制到这个文件夹
2. 在文件开头补一行说明,例如:
   `> 来源: https://example.com/article  采集日期: 2026-05-06`
3. 然后让 Claude 按 CLAUDE.md 的 Ingest Workflow 处理它
