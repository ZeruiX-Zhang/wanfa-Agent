# raw/papers/

放在这里的内容:学术论文、技术报告、白皮书。

规则:
- 文件名建议格式: `YYYY-作者-论文简称.pdf`,例如 `2017-Vaswani-AttentionIsAllYouNeed.pdf`
- 优先放 PDF 原文;如果有自己抄的关键段落,可以放 .md 副本,但不替换 PDF
- 引用论文时,wiki 层用 `[[论文短名]]` 链接到 wiki/sources/ 中的摘要页

如何放入:
1. 把 PDF 放到这个文件夹
2. 在 wiki/sources/ 让 Claude 创建一个摘要页(包含 abstract、main claims、related work)
