# raw/transcripts/

放在这里的内容:视频/播客的字幕或转录文本。

规则:
- 文件名建议格式: `YYYY-MM-DD-讲者-标题.md`,例如 `2024-04-Karpathy-MakeMore-Lecture5.md`
- 在文件开头标明: 视频链接、讲者、时长、自动转录还是人工校对
- 不要在 raw 里"修通顺",修整后的版本放 wiki/sources/

如何放入:
1. 用 YouTube/B站等平台导出字幕,或用 Whisper 等工具转录
2. 复制文本到 .md 文件
3. 让 Claude 按 CLAUDE.md 的 Ingest Workflow 处理
