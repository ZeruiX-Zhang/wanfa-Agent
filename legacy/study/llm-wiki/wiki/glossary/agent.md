# Agent

**一句话**: 能规划、调用工具、保留状态、完成多步骤任务的 LLM 应用,与"单次 prompt-response"的普通 LLM 调用相对。

**何时该用 Agent**(GPT 综述给的判断阶梯,secondhand):
普通 LLM call → RAG → tool calling → workflow → autonomous agent → 人工审批,按风险与复杂度递增。

详见 [[agent]] 概念页。
