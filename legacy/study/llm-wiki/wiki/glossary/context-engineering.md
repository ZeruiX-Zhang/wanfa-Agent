# Context Engineering

**一句话**: 管理 LLM 应用整个上下文状态的工程范式——系统指令、工具、记忆、外部数据、会话历史——而不止"写好一句 prompt"。

**对照 prompt engineering**: prompt engineering 关注一句指令的措辞;context engineering 关注模型整个"视野"的组装。

详见 [[context-engineering]] 概念页。
