# LangChain

## Summary

LangChain 是一个用于构建 LLM 应用(尤其是"上下文感知的推理应用")的开发框架。GPT 综述将其作为"基础设施型"AI 创业的代表案例。

> ⚠️ Secondhand。一手信息需要 ingest LangChain 官网或博客。

## Key People

- [[harrison-chase]] — 联合创始人 + CEO

## What It Is(secondhand)

- 框架定位:让开发者用 LLM 构建"context-aware reasoning applications"
- 提供:LLM 编排、Agent、工具调用、RAG、评估等组件
- 衍生产品/概念:LangSmith(可观测性 + evals)、LangGraph(图式 Agent 工作流)
- Agent evaluation checklist 是它的代表方法论(secondhand,待核实)

## Why It Matters

是当前 LLM 应用开发最广泛被讨论的框架之一,与 LlamaIndex 等同处第一梯队。Wiki 中所有涉及 RAG、Agent、Evals 的页面都会反复引用 LangChain 文档作为对照。
