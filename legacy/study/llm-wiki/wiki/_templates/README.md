# wiki/_templates/

Obsidian 的"模板"目录。在 Obsidian 里:**Ctrl+P → "Insert template" → 选一个**,就能把对应模板插入当前空白页。

每份模板严格对应 CLAUDE.md 里定义的页面结构。

模板列表:
- `00-concept.md` — 概念页(对应 §5)
- `01-source.md` — 来源摘要(对应 §6)
- `02-claim.md` — 原子主张(对应 §7)
- `03-contradiction.md` — 矛盾(对应 §8)
- `04-question.md` — 问题与答案
- `05-person.md` — 人物
- `06-project-or-org.md` — 项目/组织通用模板
- `07-map.md` — 主题地图

> 该目录不参与 Graph View(已在 graph.json 用 `-path:"_templates"` 排除)。
