# 金融交易 - 零基础学习路线图

> 状态：进行中  
> 目标：从零到能构建自己的交易系统并实盘操作

---

## 阶段总览

### Stage 1: 金融常识（2 周）
**验收标准**：能解释股票/债券/基金的区别，能说清市场基本运作逻辑

- [ ] 什么是金融市场？参与者有谁？
- [ ] 股票、债券、基金、衍生品的区别
- [ ] 交易所如何运作（撮合、做市）
- [ ] 基本术语：多/空、杠杆、保证金、止损止盈
- [ ] **里程碑**：能给别人讲清"股票是怎么交易的"

### Stage 2: 基本面分析（3 周）
**验收标准**：能读懂一份财报，能用 3 个指标评估一家公司

- [ ] 财务三表：资产负债表、利润表、现金流量表
- [ ] 核心指标：P/E, P/B, ROE, 毛利率、负债率
- [ ] 行业分析框架
- [ ] 护城河（Moat）概念
- [ ] **里程碑**：独立分析一只股票并写出投资逻辑

### Stage 3: 技术分析（4 周）
**验收标准**：能看懂 K 线图，能识别 3 种以上图形形态

- [ ] K 线基础：阳线、阴线、影线含义
- [ ] 均线系统：MA5, MA20, MA60 的用法
- [ ] 成交量与价格的关系
- [ ] 支撑位与阻力位
- [ ] 经典图形：头肩顶/底、双重顶/底、三角形
- [ ] 常用指标：MACD, RSI, Bollinger Bands
- [ ] **里程碑**：用 TradingView 标注一张完整的技术分析图

### Stage 4: 区块链与加密货币（3 周）
**验收标准**：能解释区块链原理，能在交易所完成基本操作

- [ ] 区块链核心原理：去中心化、共识机制、哈希
- [ ] Bitcoin vs Ethereum：区别与各自定位
- [ ] 钱包类型：热钱包 vs 冷钱包
- [ ] CEX vs DEX：中心化 vs 去中心化交易所
- [ ] DeFi 基础概念：流动性池、AMM、Yield Farming
- [ ] **里程碑**：在测试网完成一次完整交易流程

### Stage 5: 风险管理（2 周）⚠️ 最重要的阶段
**验收标准**：能设计自己的仓位管理规则，能解释为什么风控比选股重要

- [ ] 为什么大多数人亏钱？（幸存者偏差、过度交易）
- [ ] 仓位管理：固定比例法、凯利公式
- [ ] 止损策略：固定止损、跟踪止损、时间止损
- [ ] 资金曲线与回撤控制
- [ ] 交易成本计算
- [ ] **里程碑**：写出自己的风险管理清单（每次交易前必查）

### Stage 6: 交易系统构建（4 周）
**验收标准**：能用回测验证一个交易策略的有效性

- [ ] 什么是交易系统？入场、出场、仓位、规则
- [ ] 趋势跟踪 vs 均值回归
- [ ] 回测基础（可用 Python，与编程交叉！）
- [ ] 模拟盘实践
- [ ] 交易日志的写法与复盘
- [ ] **里程碑**：用模拟盘跑完一个月，写出完整复盘报告

## 金融领域特殊规则

### 验收方式扩展
| 标准 | 含义 | 验证方式 |
|------|------|----------|
| **能解释** | 用自己的话讲清金融概念 | 闭卷复述 |
| **能判断** | 面对具体案例能做出合理判断 | 案例分析 |
| **能执行** | 在交易平台上完成实际操作 | 模拟盘操作记录 |
| **能复盘** | 事后分析对错并提取教训 | 交易日志 |

### 红线（金融特有）
- **不要用 AI 做交易决策**——AI 可以帮你学习，但买卖决定必须自己做
- **不要跳过 Stage 5（风控）**——这是亏钱和不亏钱的分界线
- **先模拟盘，至少 1 个月后再考虑实盘**
- **实盘初期只用你能承受完全亏损的金额**

### 学习节奏建议
- 理论 : 案例分析 : 模拟实操 = 4 : 3 : 3
- 每学一个概念，找 3 个真实市场案例验证
- 从 Stage 3 起，每天花 10 分钟看盘培养盘感
