# raw/english/ - 英语学习原始材料

> 重点方向：技术文档/论文阅读能力。材料按类型存放。

## 建议分类存放

```
english/
├── 01-vocabulary/        ← 词汇表、词根词缀、高频技术词
├── 02-grammar/           ← 语法规则、长难句分析
├── 03-tech-docs/         ← 技术文档原文（Python docs, RFC 等）
├── 04-papers/            ← 学术论文原文
├── 05-reading-logs/      ← 精读记录：生词、句式、理解笔记
└── 06-writing/           ← 英文写作练习（commit message, README 等）
```

## 推荐材料来源
- Python/JavaScript 官方文档（边学编程边练英语）
- arXiv 论文（和编程领域交叉）
- Hacker News 文章
- 技术博客：Martin Fowler, Joel on Software
- Investopedia 文章（和金融领域交叉）
