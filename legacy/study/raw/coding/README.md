# raw/coding/ - AI 编程与大模型原始材料

> 方向：AI 辅助编程 + 大模型原理/算法研究  
> Python 有基础，目标是能读论文、复现、改进模型。

## 建议分类存放

```
coding/
├── 01-python-for-ai/         ← Python 进阶（NumPy, 数据处理, OOP）
├── 02-math-foundations/       ← 数学基础（线性代数, 概率, 微积分, 优化）
├── 03-ml-basics/              ← 机器学习基础（经典算法, 损失函数, 评估）
├── 04-deep-learning/          ← 深度学习（神经网络, CNN, RNN, 训练技巧）
├── 05-transformer-llm/        ← Transformer & LLM（注意力, 预训练, RLHF）
├── 06-training-infra/         ← 训练基础设施（分布式, 混合精度, DeepSpeed）
├── 07-fine-tuning/            ← 微调技术（LoRA, QLoRA, PEFT, SFT, DPO）
├── 08-inference-deploy/       ← 推理与部署（量化, vLLM, TensorRT）
├── 09-ai-agents/              ← AI Agent（工具调用, RAG, 多智能体）
├── 10-ai-coding-tools/        ← AI 编程工具（Cursor, Copilot, Claude Code）
├── 11-papers/                 ← 论文原文
└── 12-projects/               ← 项目实战记录
```
