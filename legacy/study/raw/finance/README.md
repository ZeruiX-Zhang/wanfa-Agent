# raw/finance/ - 金融交易原始材料

> 零基础 → 实战交易路线，材料按阶段放入对应子文件夹。

## 建议分阶段存放

```
finance/
├── 01-basics/            ← 金融市场基础、术语、市场结构
├── 02-stocks/            ← 股票：基本面分析、技术分析
├── 03-crypto/            ← 区块链原理、加密货币交易
├── 04-technical-analysis/← K线、指标、图形、量价关系
├── 05-risk-management/   ← 仓位管理、止损、资金曲线
├── 06-trading-systems/   ← 交易系统构建、回测
├── 07-psychology/        ← 交易心理、行为金融
└── 08-trade-journal/     ← 实盘/模拟盘交易记录
```

## 推荐材料来源
- 《聪明的投资者》《股票大作手回忆录》
- Investopedia (英文，顺便练英语)
