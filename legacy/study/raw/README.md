# raw/ - 原始材料层

> **这个文件夹里的文件永远不修改。**
> 这是你的"真相层"，所有 wiki 页面都必须回指这里的来源。

## 如何使用

1. 把学习材料放进对应领域的子文件夹：
   ```
   raw/
   ├── coding/     ← AI 编程与大模型（按阶段细分，见 coding/README.md）
   ├── finance/    ← 金融交易（按阶段细分，见 finance/README.md）
   └── english/    ← 英语材料（按类型细分，见 english/README.md）
   ```
2. 文件命名建议：`简短描述-来源.格式`
3. 放进来之后**不要修改原始文件**，所有笔记和理解都写在 `wiki/` 中
